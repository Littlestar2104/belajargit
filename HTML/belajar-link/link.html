Belajar Relative URL

  • hello.html
  • akbar/index.html
  • /color.html

Penjelasan

File hello.html berada di tempat yang sama (folder yang sama) dengan file HTML ini. Jadi, link ini akan mencari file hello.html di folder yang sedang digunakan.

File index.html berada di dalam folder bernama akbar, yang ada di dalam folder tempat file HTML ini berada. Jadi, link ini akan masuk ke folder akbar terlebih dahulu, lalu mencari file hello.html.

File color.html berada di folder utama (root direktori) server. Jadi, link ini akan langsung mencari file link.html di bagian paling atas dari semua folder di server.
